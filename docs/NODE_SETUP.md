## Token Distribution

| Allocation | Percentage |
|------------|------------|
| Community Contribution Pool | 60% |
| Validator Rewards | 20% |
| Protocol Treasury | 15% |
| Protocol Reserve | 5% |

Ownership of the network is earned through **contribution instead of speculation**.
